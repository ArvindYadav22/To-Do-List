# TodoList – React + Vite + TailwindCSS

A simple Todo List application built using React, Vite, and TailwindCSS.  
The project includes ESLint configuration, PostCSS, and a clean development setup.

## Features

- Fast Vite development server
- TailwindCSS styling
- React 19 with hooks
- ESLint with React Hooks and React Refresh plugins
- Optimized production build
- Simple and clean project structure

## Installation

1. Clone the repository:
git clone <your-repo-url>
cd todolist

2. Install dependencies:
```
npm install
```

## Running the Project

Start the Vite development server:
```
npm run dev
```


Open in browser: http://localhost:5173

## Production Build

Build the production version:
```
npm run build
```

Preview the output:
```
npm run preview
```

## TailwindCSS Setup

The project includes:

- `postcss.config.js` for Tailwind and autoprefixer  
- `tailwind.config.js` for content paths and theme configuration  

## ESLint Setup

ESLint includes:

- Recommended JavaScript rules  
- React Hooks plugin  
- React Refresh plugin for Vite  

Run linting:

npm run lint
